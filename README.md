# Smart Home Security System Using Automata

An academic project that applies concepts of Automata Theory and Formal Languages to a Smart Home Security System integrated with Face Recognition authentication.

---

## Overview

This project demonstrates how finite automata, grammar rules, lexical analysis, and syntax analysis can be applied in a real-world inspired smart home security scenario.

The system simulates a secure authentication process where authorized users can access a smart home environment using face recognition validation and formal language processing techniques.
